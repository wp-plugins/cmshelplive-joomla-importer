Import articles from your Joomla site into WordPress with few simple steps.

== Description ==

Those who are looking to migrate from Joomla to WordPress, this easy to use plugin allows importing Joomla content into their WordPress sites. It will import both Joomla articles and multimedia. 


== Installation ==

1. Download CMSHelpLive: Joomla to WordPress Importer Pro

2. Go to your Dashboard and locate Plugins tab. Click on Upload Plugin and specify the path of downloaded plugin in the text box.

3. Click on Activate Plugin



== Frequently Asked Questions ==

= Is there a limit to the number of articles I can import? =

No, there is no limit. You can import all the articles.

= Where are the articles imported in WordPress? =

But default articles are imported into posts. But you also have the option to import as pages.

= Does it also import Joomla extension data? =

No, it only imports from Article manager component of Joomla.
